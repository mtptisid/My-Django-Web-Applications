Django Installation Guide

Installing Python

Linux

sudo apt update
sudo apt install python3 python3-pip

Windows

Download Python installer from Python.org. Run the installer, ensure "Add Python to PATH" is checked, and proceed with installation.

Installing Django

pip install django

Setting up a Django Project

mkdir myproject
cd myproject
django-admin startproject myproject

Running Django Server

python manage.py runserver

Open your web browser and go to http://127.0.0.1:8000/ to see your Django website.
